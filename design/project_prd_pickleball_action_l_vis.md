# Project Brief: Pickleball Action Lévis Mobile App

## 1. Project Overview
Pickleball Action Lévis is a mobile application (iOS & Android) designed to streamline the organization and management of pickleball sessions for the club. The app bridges the gap between pre-session registration and live on-court coordination, specifically focusing on session tracking, player matrix generation, and real-time game management.

## 2. Target Audience
- **Club Members:** Players looking to view upcoming sessions and track their registration status.
- **On-Court Coordinators:** Volunteers or staff responsible for organizing players into matches, managing attendance, and tracking game time.

## 3. Core Features & User Flows

### A. Onboarding & Profile
- **Level Selection:** New users select their skill level (Beginner, Intermediate, Pro) during onboarding to ensure appropriate session filtering.
- **Profile Management:** Users can update their skill level and preferences at any time.
- **Feedback System:** A dedicated form for users to provide suggestions and feedback specifically about the application's performance and features.

### B. Session Agenda
- **Registration Tracking:** Clear visual distinction between sessions where a player is "Registered" (Inscrit) versus available sessions.
- **Session Details:** View time, location, and current player count for daily and upcoming sessions.
- **Filtering:** Ability to filter the agenda based on the user's skill level.

### C. Matrix Generator (Session Setup)
- **Attendance Management:** Coordinators can confirm presence from a pre-registered list, add guests on the fly, and remove players with a confirmation alert.
- **Batch Addition:** High-speed player addition interface allowing for batch selection of existing members or manual entry of guests.
- **Court Configuration:** Adjustable setting for the number of available courts to determine match capacity.

### D. Game Management (Live Rotation)
- **Round-Based Matchmaking:** Automated pairings for multiple rounds of play.
- **Live Timer:** A prominent countdown timer for active rotations.
- **Court Assignments:** Clear "Team A vs Team B" view for each active court.
- **Waiting Area:** Real-time visibility into which players are currently on the bench ("Salle d'attente").
- **Flexible Management:** Ability to modify the player list or skip/pause rounds during a live session.

## 4. Design & Visual Identity
The app uses a high-contrast, modern aesthetic centered around the Pickleball Action Lévis brand.

- **Primary Colors:** Brand Orange (#ff5700) for actions and highlights, paired with deep neutrals.
- **Typography:** Lexend (Primary) and Space Grotesk for a tech-forward, readable feel.
- **Visual Styles Explored:**
    - **Midnight Pro (Preferred):** Deep navy backgrounds with subtle gradients and orange accents.
    - **High Contrast Dark:** Pure black backgrounds optimized for outdoor OLED visibility.
    - **Glassmorphism:** Premium look using background blurs and transparency.

## 5. Technical Constraints
- **Language:** Fully localized in French.
- **Platform:** Cross-platform mobile (iOS/Android).
- **Architecture:** Designed for seamless transition between player list management and live game state.

## 6. Key Screens Established
1. Onboarding: Level Selection
2. Sessions Agenda (Multiple variants)
3. Profile & Settings
4. Matrix Setup / Attendance Management
5. Add Players (Batch & Guests)
6. Game Management (Live Rotation & Timer)
7. Feedback Form
